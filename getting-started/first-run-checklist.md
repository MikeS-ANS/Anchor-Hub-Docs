# First-Run Checklist

Complete these steps when setting up Anchor Hub for the first time.

## Installation

- [ ] Download the latest installer from the [Anchor Hub releases page](https://github.com/MikeS-ANS/Anchor-Hub/releases/latest)
- [ ] Run `Anchor-Hub-Setup-X.X.X.exe` — it installs automatically, no options required
- [ ] Launch Anchor Hub from the Start Menu

## Login

- [ ] Click **Sign in with Microsoft**
- [ ] Sign in with your `@anchornetworksolutions.com` account
- [ ] Confirm the app loads and shows your name/photo in the top-right corner

## Autotask Write Key (required for pushing to contracts)

- [ ] Open **Settings** (gear icon, bottom-left)
- [ ] Enter your Autotask API Username, API Key, and Integration Code
- [ ] Click **Save**

> If you don't have an Autotask API user yet, skip this for now and contact Mike.

## Verify Everything Works

- [ ] Navigate to **Invoice Processor**
- [ ] Click **Load Most Recent Invoice**
- [ ] Confirm the invoice loads and the breakdown table appears
- [ ] If you set up your AT key: confirm the **Push to Autotask** button is visible

## You're All Set

Once the invoice loads successfully, Anchor Hub is fully configured. The app updates itself automatically — you'll see a prompt when a new version is available.
