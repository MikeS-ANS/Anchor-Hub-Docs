# Welcome to Anchor Hub

Anchor Hub is Anchor Network Solutions' internal tool for managing recurring billing across Pax8, Autotask PSA, and other platforms. It replaces manual copy-paste workflows with direct integrations so the team spends less time reconciling spreadsheets and more time with clients.

## What Anchor Hub Does

| Tool | What it does |
|---|---|
| **Invoice Processor** | Loads a Pax8 invoice, breaks down costs by service (Azure, Nerdio, Exclaimer, etc.), and pushes updated pricing and seat counts directly to Autotask contracts |
| **Subscription Audit** | Compares active Pax8 subscriptions against Autotask contract services and flags quantity mismatches |
| **Margin Analyzer** | Reviews Azure partner costs vs. billed prices across all clients |
| **Company Mapping** | Links Pax8 company names to their Autotask counterparts — the foundation that makes all the other tools work |
| **Contract Changes** | Tracks changes to contract services over time |
| **Duo Management** | Manages Duo Security sub-accounts and server enrollment |

## Who Uses It

Anchor Hub is for internal ANS staff only. Access requires an `@anchornetworksolutions.com` Microsoft account.

Different roles see different features:
- **Standard** — read-only access to reports and breakdowns
- **IT** — can run audits and view invoice data
- **Admin** — full access including pushing updates to Autotask
