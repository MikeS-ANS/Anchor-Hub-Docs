# v1.4.2 — Fuzzy Company Auto-Match & Push History

**Released:** June 19, 2026

## What's New

### Fuzzy Company Auto-Match
The Invoice Processor now automatically finds Autotask matches for unmapped Pax8 clients when you load an invoice — no manual mapping required for most clients.

- High-confidence matches (≥85%) are saved silently and push buttons work immediately
- Lower-confidence matches appear in a yellow confirmation panel — click **Confirm** to accept
- Confirmed matches are saved permanently and apply to all future invoices

### Push History
Every Autotask push is now logged so you can see exactly what happened and identify anything that needs fixing.

- Push History table appears at the bottom of invoice results after every load
- Click any row to expand per-client detail
- Shows success, no-change, no-mapping, no-contract, and error status per client
- History is stored locally and shows the last 20 runs

## Also in This Build

- Pax8 API credentials migrated to Azure Key Vault (no longer stored per-machine)
- Autotask two-tier credential model: shared read-only credentials from Key Vault, personal write key from Settings
- Invoice loader improvements: "Load Most Recent Invoice" button, month names instead of dates (June 2026), dollar totals in the past invoices dropdown
- Fixed: AT push modal was transparent on some machines

## How to Update

The app updates automatically. If you haven't seen the update prompt yet, restart Anchor Hub.
