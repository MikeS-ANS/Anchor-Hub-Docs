# v1.4.1 — Duo Management & Microsoft SSO

**Released:** June 2026

## What's New

### Duo Management
New tool for managing Duo Security sub-accounts and server enrollment.

- **New Sub Account wizard** — create and configure a Duo sub-account for a new client
- **Add Duo to Server** — enroll a server in an existing Duo account via Datto RMM Quick Job (no manual RDP required)
- Excluded accounts list to hide internal/irrelevant Duo accounts

### Microsoft SSO Login
Anchor Hub now uses your `@anchornetworksolutions.com` Microsoft account for login — no separate credentials needed.

- Sign in once and stay signed in
- Profile photo pulled from Microsoft automatically
- Role-based access: admin, IT, standard, and read-only tiers

### Other Improvements
- Contract Name column added to relevant tables
- Logo display fixes
- Azure enterprise infrastructure groundwork (Key Vault, Entra ID roles)

## How to Update

The app updates automatically. If you haven't seen the update prompt yet, restart Anchor Hub.
