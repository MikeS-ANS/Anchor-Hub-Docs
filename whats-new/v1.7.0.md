# v1.7.0 — Meraki Admin Management

## Meraki Admin Management — Major Update

This release is a comprehensive overhaul of the Meraki Admin Management tool, making it practical to audit and remediate admin gaps across all client organizations at scale.

---

### Audit Tab Redesign

The old view expanded all 70+ orgs at once, which was unusable. The new view is a compact table with gap analysis against the **Organization Template** org as your baseline.

**At a glance you can see:**
- How many ANS admins each org has vs. the baseline
- How many admins are **missing** (in template but not in the org)
- How many admins are **extra** (in the org but not in the template)
- Overall status per org (Baseline / ⚠ issues / ✓ OK)

Click any row to expand it and see the full detail.

**New audit features:**
- **Filter box** — type to search orgs by name
- **Issue count badge** — the Audit tab shows `⚠15` when gaps exist
- **Last refreshed** timestamp in the summary bar
- **Export CSV** — exports the full audit table (org, ANS count, missing, extra, status)

---

### Fixing Gaps

**Per-org:**
- **Missing ANS Admins** section shows each missing admin as a chip with a green **+ Add** button
- **+ Add All Missing** button adds every missing admin to that org in one click
- **Extra ANS Admins** section shows admins not in the baseline with an amber **Remove** button per chip
- **Remove All Extra** bulk removes all non-baseline ANS admins from that org
- All remove actions require a **two-click confirm** to prevent misclicks

**Bulk — across all orgs:**
- **Fix All Missing** button in the toolbar adds every missing ANS admin across every org with gaps in one operation. Requires confirmation before running.

---

### Per-Org Admin Management

Expanding an org row now shows full admin management controls:

- **Remove** button on every admin row (requires two-click confirm)
- **+ Add Admin to this org** inline form — enter email and name to add a one-off admin to just this org with full access

---

### Excluded Orgs Tab

For organizations that ANS has been offboarded from but hasn't been removed from yet:

- Mark any org as **Excluded** from the audit row
- Excluded orgs are hidden from the Audit tab and skipped during bulk add/remove operations
- The **Excluded Orgs** tab provides a separate audit trail with **Un-exclude** capability

---

### Company Mapping — Meraki Orgs Tab

A new tab in the Company Mapping tool maps Meraki org names to their AT companies (so the correct client name appears in the audit).

- **Auto-Match** button runs fuzzy scoring against all AT companies and suggests matches
- Each suggestion shows a **confidence badge** (High ≥75% / Med / Low)
- **Accept** or **Skip** individual suggestions, or **Accept All High Confidence** in one click
- Manual **+ Assign** search still available for orgs the auto-match doesn't cover
