# v1.5.9 — Centralized Hub Company Directory

## Centralized Hub Company Directory

All tools that map client names to Autotask companies now share a single hub directory stored on SharePoint instead of maintaining separate mapping files per tool.

**What changed:**
- All company mappings live in one file: `ANS-Company Shared → Anchor Hub → hub-company-mappings.json`
- **Blackpoint** migrated from its own `bp-company-mappings.json` to the central hub
- **Company Mapping** reads and writes the central hub directly
- **Kaseya Invoice Processor** already uses the central hub (introduced in v1.5.8)

**Why it matters:**
- Confirm or exclude a company in any tool and every other tool immediately sees it
- Single source of truth — no more duplicate or conflicting mappings across tools
- New tools added in the future automatically benefit from existing confirmed mappings

This is an internal infrastructure change. The UI for each tool looks and works the same — the hub directory just runs behind the scenes.
