# v1.5.7 — Home Screen, Help Page & Bug Fixes

## Home Screen (Daily Start Page)

The Home screen has been rebuilt as a daily start page. When you open Anchor Hub, you'll now see:

- **Greeting bar** — your name, avatar, and today's date
- **Quick Links** — one-click tiles to your most-used tools and portals. Admin-managed links come from SharePoint; you can also add your own personal links with the **+ Add link** button
- **Today's Calendar** — your Outlook calendar events for today, pulled directly from Microsoft 365
- **Autotask Tickets** — a priority snapshot of your assigned tickets:
  - Critical & High count (highlighted in red when non-zero)
  - Due Today count
  - Overdue count
  - Total Assigned count
  - Your top 5 tickets listed by urgency with company name

The existing tool cards grid is still accessible via the **Tools** item in the sidebar.

## Help Page

- **Request Support** and **Report a Bug** buttons are now in the page header — visible immediately without scrolling
- **Request Support** sends a message directly to your admin via Microsoft 365 — no email client required

## Bug Fixes

- Calendar events now display in your local system timezone (previously could show in the wrong timezone)
- User dropdown card was appearing transparent — fixed
- Autotask portal link now opens the correct portal URL
- Autotask ticket widget now correctly resolves staff accounts regardless of whether the login email uses `@anchornetworksolutions.com` or `@anchorns.com`
