# v1.5.0 — Modular Architecture

**Released:** June 2026

## What's New

### Under the Hood — Modular Refactor

This release is an internal architecture improvement with no user-facing changes. All tools work exactly as before.

`main.js` has been split from ~3,400 lines into 14 focused modules. Each tool's backend logic now lives in its own file, making future updates faster, safer, and easier to maintain.

**What this means for you:** Nothing changes day-to-day. You won't notice a difference in how the app looks or works. The benefit shows up as faster delivery of new features going forward.
