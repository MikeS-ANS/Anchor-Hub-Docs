# v1.4.3 — Project Profitability

**Released:** June 2026

## What's New

### Project Profitability Tool
New tool for analyzing project margins directly from Autotask — no spreadsheets required.

- Pull all completed (or active) projects for any date range
- See gross margin, cost of delivery, billed vs. estimated hours, and billing efficiency per project
- Color-coded margin indicators: green = healthy, amber = watch, red = below threshold
- Automatically flags low-margin projects, unbilled T&M hours, pending milestones, and missing hour estimates
- Sortable 16-column table — sort by margin, company, lead, or any other field
- Configurable blended labor rate, standard bill rate, and margin warning threshold
- Export to Excel: 3 sheets covering Project Summary, Flags & Issues, and Lead Summary

See the [Project Profitability guide](../tool-guides/project-profitability.md) for full details.
